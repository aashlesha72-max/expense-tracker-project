# Expense Tracker Project

## Project Overview
This is a Python-based Command Line Expense Tracker application developed as part of my internship project.  
The system allows users to manage their daily expenses efficiently using JSON for data storage.

## Features
- Add new expense
- View all expenses
- Update existing expense
- Delete expense
- Generate expense reports
- Calculate total spending

## Technologies Used
- Python
- JSON
- File Handling
- Modular Programming

## Project Structure
- main.py – Controls program flow and menu
- firstcode.py – Contains main expense functions
- database.py – Handles data storage and retrieval
- reports.py – Generates expense reports
- expenses.json – Stores expense data

## How to Run the Project
1. Install Python
2. Open terminal in the project folder
3. Run the command: python main.py
